# OSCP Preparation Repository 🛡️

This repository contains my personal notes, custom scripts, exploits, and lab setups used while preparing for the [Offensive Security Certified Professional (OSCP)](https://www.offensive-security.com/pwk-oscp/) certification.

## 🧰 Contents

- `notes/` – Topic-wise study notes and enumeration checklists.
- `scripts/` – Custom privilege escalation and enumeration scripts.
- `exploits/` – Proof-of-concept code and modifications of public exploits.
- `labs/` – Setup instructions and vulnerable VM configurations for practice.
- `walkthroughs/` – Write-ups and solutions for OSCP-style boxes and labs.

## 🧪 Technologies Covered

- Linux privilege escalation
- Windows privilege escalation
- Buffer overflow exploits
- Web application attacks
- Active Directory enumeration
- Exploitation methodologies and post-exploitation

## 🚨 Disclaimer

All content in this repository is for educational and ethical hacking purposes only. Use responsibly and only on systems you own or have explicit permission to test.
